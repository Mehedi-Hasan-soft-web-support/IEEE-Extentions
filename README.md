## ✨ বৈশিষ্ট্য (Features)

- 📄 **PDF Tools** - নিরাপদ PDF সম্পাদনা এবং রূপান্তর
- 🔗 **VTOOLS Integration** - IEEE VTOOLS এ দ্রুত অ্যাক্সেস
- 📋 **SAR Form** - সহজ SAR ফর্ম সম্পূর্ণকরণ
- 🔒 **100% Local** - সব ডেটা আপনার ডিভাইসে থাকে
- ⚡ **No Server Required** - সম্পূর্ণ স্বাধীন এবং দ্রুত
- 🎯 **IEEE Official** - IEEE এর সরকারী সহযোগী

## ⚡ দ্রুত ইনস্টলেশন (Quick Install)

### একটি ক্লিকে ইনস্টল করুন:

> **নোট:** নিচের পদ্ধতি সবচেয়ে সহজ। শুধু "**ডাউনলোড এবং লোড**" করুন।

#### **পদ্ধতি ১: সরাসরি ডাউনলোড করুন** ⬇️

1. **ZIP ফাইল ডাউনলোড করুন:**
   - [📥 IEEE-Extentions-main.zip ডাউনলোড করুন](https://github.com/Mehedi-Hasan-soft-web-support/IEEE-Extentions/raw/main/IEEE-Extentions-main.zip)

2. **ZIP এক্সট্র্যাক্ট করুন:**
   - যেকোনো ফোল্ডারে ডাউনলোড করা ZIP ফাইল আনজিপ করুন

3. **Chrome খোলুন এবং এতে যান:**
   ```
   chrome://extensions/
   ```

4. **Developer Mode চালু করুন:**
   - উপরে ডানদিকে "Developer mode" টগল **চালু** করুন

5. **"Load unpacked" ক্লিক করুন:**
   - আনজিপ করা ফোল্ডার সিলেক্ট করুন
   - **Select Folder** ক্লিক করুন

6. ✅ **সম্পন্ন!** Extension এখন ইনস্টল হয়েছে

---

#### **পদ্ধতি ২: স্বয়ংক্রিয় স্ক্রিপ্ট ব্যবহার করুন** (সহজতম)

##### **Windows ব্যবহারকারীদের জন্য:**
```batch
1. install-extension-windows.bat ডাউনলোড করুন
2. ফাইলে রাইট ক্লিক করুন → "Run as Administrator"
3. স্ক্রিপ্ট স্বয়ংক্রিয়ভাবে সবকিছু করবে
```

##### **macOS ব্যবহারকারীদের জন্য:**
```bash
1. install-extension-mac.sh ডাউনলোড করুন
2. Terminal খোলুন এবং এটি চালান:
   chmod +x install-extension-mac.sh
   ./install-extension-mac.sh
```

##### **Linux ব্যবহারকারীদের জন্য:**
```bash
1. install-extension-linux.sh ডাউনলোড করুন
2. Terminal খোলুন এবং এটি চালান:
   chmod +x install-extension-linux.sh
   ./install-extension-linux.sh
```

## 🆘 সমস্যা সমাধান (Troubleshooting)

### সাধারণ সমস্যা এবং সমাধান:

#### ❌ "Manifest not found" ত্রুটি
**সমাধান:** নিশ্চিত করুন যে `manifest.json` ফাইল সরাসরি নির্বাচিত ফোল্ডারে আছে

#### ❌ Extension লোড হয়নি বা দৃশ্যমান নয়
**সমাধান:** Chrome সম্পূর্ণভাবে বন্ধ করুন এবং পুনরায় খুলুন

#### ❌ Developer Mode খুঁজে পাচ্ছেন না
**সমাধান:** `chrome://extensions/` এ যান, টগল উপরে ডানদিকে

#### ❌ Extension আইকন নেই
**সমাধান:** Chrome টুলবারে এক্সটেনশন আইকন (পাজল) ক্লিক করুন এবং "Show" নির্বাচন করুন

**আরও সাহায্যের জন্য:** [📚 সম্পূর্ণ গাইড দেখুন](./INSTALLATION_GUIDE.md#সমস্যা-সমাধান-troubleshooting)

## 🔄 আপডেট এবং রক্ষণাবেক্ষণ

### নতুন সংস্করণ ইনস্টল করতে:
1. রিপোজিটরি থেকে সর্বশেষ ফাইল ডাউনলোড করুন
2. `chrome://extensions/` এ যান
3. পুরাতন এক্সটেনশন সরান (Delete)
4. নতুন ফোল্ডার লোড করুন

### Extension আনলোড করতে:
1. `chrome://extensions/` এ যান
2. "SIGHT Tools" খুঁজুন
3. **Remove** ক্লিক করুন
